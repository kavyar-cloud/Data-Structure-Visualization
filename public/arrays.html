Arrays

What is an Array?

An array is a collection of elements of the same type stored at contiguous memory locations. It allows you to store multiple values under a single variable name and access them using an index.

Operations Explained:

  • Insert: Add a new element at a specific index in the array.
  • Delete: Remove an element from a specific index.
  • Length: Get the number of elements currently in the array.
  • Display: Show all elements of the array in order.
