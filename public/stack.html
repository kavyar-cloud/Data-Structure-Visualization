What is a Stack?

A Stack is a linear data structure that follows a particular order in which the operations are performed. The order is LIFO(Last In First Out). LIFO implies that the element that is inserted last, comes out first.

  • Push: Insert an element into the stack.
  • Pop: Remove the top element.
  • Peek: View the top element.
  • Display: Show all stack elements.
